Web App Designer




WAD.js or Web app designer is a javascript library that is aimed to simplify and save time on designing Componets like Forms,Sidebar,Tooltiops... and more, With this library we believe developers will complete thier dream projects without wasting any time and, WAD.js is a game-changing when it comes to responsive designs, as we all know "Responsive" is one of the scary words to all developers, guess what!!? with WAD.js the nightmare of not being able to make anything responsive it is now gone
